# Scientific Calculator Application using Python and Tkinter

## Diploma CST 4th Semester Minor Project

---

## Project Overview

Scientific Calculator Application is a desktop-based calculator developed using **Python** and **Tkinter**.

The application performs both basic arithmetic and scientific calculations through a simple graphical user interface (GUI). It is designed as a Diploma 4th Semester Minor Project with emphasis on clean code, beginner-friendly implementation, and easy maintenance.

## Error Handling

The calculator safely handles common user mistakes.

Examples

* Empty Expression
* Invalid Expression
* Division by Zero
* Invalid Scientific Function
* Math Domain Errors
* Incorrect Factorial Input
* Invalid Keyboard Input

Instead of crashing, the calculator displays an error message.

---

# Security

This project does **not** use Python's unsafe `eval()` function directly.

Instead, it safely evaluates expressions using Python's **AST (Abstract Syntax Tree)** module.

Only approved mathematical operators and functions are allowed.

This prevents execution of arbitrary Python code.
